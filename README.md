# 🚀 QuickSnip — Modern Code Snippet Manager

QuickSnip is a sleek and developer-friendly web application built to help programmers **save, organize, search, copy, and share reusable code snippets effortlessly**.

Designed with a clean SaaS-inspired interface, QuickSnip improves productivity by giving developers one centralized workspace for frequently used code blocks.

---

## ✨ Live Features

* ✅ Save custom code snippets instantly
* ✅ Categorize snippets by programming language
* ✅ Search snippets by title
* ✅ Filter snippets by language
* ✅ Copy snippets to clipboard with one click
* ✅ Share snippets directly to WhatsApp
* ✅ Local storage persistence (no backend required)
* ✅ Dark / Light mode support
* ✅ Responsive premium UI
* ✅ Contact & legal support pages
* ✅ Modern 404 handling

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/rkgupta-dev/code-snippet-manager.git
```

Move into project folder:

```bash
cd quicksnip
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run serve
```

Build for production:

```bash
npm run build
```
